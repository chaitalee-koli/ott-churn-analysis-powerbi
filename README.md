# ott-churn-analysis-powerbi
## Project Overview
This project analyzes OTT platform customer churn using Power BI. 
The dashboard provides insights into subscriber behavior, churn patterns, and retention trends.

## Tools Used
- Excel / CSV Dataset
- SQL
- Power BI
- Data Cleaning

## Key KPIs
- Total Subscribers
- Churn Rate
- Active vs Inactive Users
- Monthly Revenue

## Key Insights
- Customers with shorter subscription duration have higher churn.
- Premium plan users show better retention.
- Certain regions show higher churn rates.

## Dashboard Features
- Interactive filters
- KPI cards
- Trend analysis
- Customer segmentation
