OTT Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn behavior for an OTT platform using Power BI, SQL, and Excel.
The goal is to identify key factors influencing customer cancellations and provide actionable insights to improve retention and revenue.

## Tools Used
-Power BI (Dashboard & Visualization)
-SQL (Data Extraction & Querying)
-Excel (Data Cleaning & Preprocessing)

## Key KPIs
- Total Subscribers
- Churn Rate
- Active vs Inactive Users
- Revenue Lost
  
## Key Insights
1. Customer Churn Overview
-Displays total churned customers and churn rate
-Tracks churn trends across different years

2. Customer Method Risk
-Analyzes churn based on payment methods
-Identifies high-risk payment categories contributing to churn

3. Risk by Age Group
-Compares churn across different age segments
-Helps identify which age group is more likely to leave

4. Churn by Year
-Shows yearly churn trends
-Helps detect patterns and seasonal behavior

5. Churn by Device
-Analyzes which devices (Mobile, TV, Laptop, etc.) have higher churn rates

6. State-wise Churn Analysis
-Visualizes geographic distribution of churn
-Identifies high-risk regions

## Dashboard Features
- Interactive filters
- KPI cards
- Trend analysis
- Customer segmentation
